IOTA Developers Education

Skills you can apply - that's what you'll learn in these free micro-courses. They're the fastest (and most fun) way to learn IOTA.

Programming knowledge of NodeJS or Python plus experience installing open-source software is recommended


IOTA DEVELOPER MICRO-COURSES

IOTA Basics

Learn new concepts essential to start using an IOTA network

Transactions

Learn how to send and receive transactions

Tangle Technology

Understand the data structure that's formed by the connections among transactions in IOTA's distributed ledger

IRI

The IRI (IOTA reference implementation) is open-source Java software that defines the IOTA protocol

Hub

Hub is a multi-user-wallet offering an easy way to manage deposits and withdrawals of IOTA tokens

Compass

Compass is an open-source Coordinator that can be used to protect an IOTA network against attacks

Data Marketplace

IOTA Data Marketplace is an open-source data marketplace for publishing or selling sensor data
