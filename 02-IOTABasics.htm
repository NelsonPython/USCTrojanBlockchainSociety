Knowledge Check


1. What is an IRI Node?
(a) IOTA tokens that have value
(b) IOTA reference implementation that defines the IOTA protocol
(c) seed generators
(d) a smart contract between two or more parties


2. What is a seed?
(a) accounts from which transactions are sent and received
(b) a secret password that's used to create addresses and to sign bundles whose transactions withdraw IOTA tokens
(c) a password that can be reset by the helpdesk if you lose it
(d) any combination of letters and numbers

(a)SENDER ONE 9999999999999999999999999999999999999999999999999999999999999999999999
(b)SENDERSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSS
(c)SENDERsssssssssssssssssssssssssssssssssssssssssssssssssssssssssssssssssssssssssss
(d)1SENDERSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSSS


4. According to the docs, what are two basic types of transactions?
(a) Bundles and seeds
(b) Debit and credit
(c) Withdraw and deposit
(d) Input and output


5. IRI nodes are the core of an IOTA network, and are responsible for which key functions:
(a) Validate transactions
(b) Store valid transactions in a ledger
(c) Allow clients to interact with the IRI and have their transactions appended to the ledger
(d) All of the above


6.What is the Tangle?
(a) accounts from which transactions are sent and received
(b) the reference implementation that defines the IOTA protocol
(c) the data structure that's formed by the connections among a node's transactions
(d) a secret password that's used to create addresses and to sign bundles whose transactions withdraw IOTA tokens


7. Transactions in the Tangle can be in one of two states:
(a) Input or output
(b) Seed or address
(c) Pending or confirmed
(d) Atomic or individual
