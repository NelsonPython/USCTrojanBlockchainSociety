Coffee farm water conservation lab

To grow coffee, farmers can use devices to monitor soil, sunlight, and water. This water conservation lab demonstrates a device with sensors to monitor plants that uses IOTA tokens to buy water when the plants are thirsty. It is useful for learning machine-2-machine transactions. You will need a Raspberry Pi Zero W plus a rain sensor and a soil moisture sensor














Raspberry Pi Zero W
For this lab, you need a Raspberry Pi Zero W Mega Kit with GPIO header attached. To connect directly to your Raspberry Pi Zero W, you will also need a miniHDMI > HDMI adaptor and a microUSB > USB adaptor. Power Raspberry Pi with a wall plug or a USB battery capable of powering mobile phones.

Raspbian Operating System
The Mega Kit includes a 16GB MicroSD with the OS installed. You can also download and install Raspbian

Interacting with Raspberry Pi Zero W
There are two ways to interact with Raspberry Pi: directly and over SSH using software like PuTTY. When deciding how to connect directly, remember Raspberry Pi Zero W only has one USB port. It is possible to switch between a USB mouse and a separate USB keyboard but it is clumsy.
